# HTTP Decoy

## Usage

```sh
Usage of /bin/http-decoy:
  -basic-auth string
    	Set basic auth realm
  -host string
    	Host address to listen to (default "0.0.0.0")
  -log-credentials
    	Log credentials
  -port int
    	Port number to listen to (default 8080)
  -server-header string
    	Set server header
```
